# Video Haven

A free and open source video editor for web, desktop, and mobile.

## Status
The project is currently under development, with a focus on creating a plugin-first architecture and a more intuitive user interface.

## What I changed
I added a new plugin system, which allows developers to create and share custom plugins for the editor. I also simplified the user interface and improved the overall performance of the application.

## Install and Usage
To get started, install the dependencies with `bun install`, then run `moon run web:dev` to start the development server.
